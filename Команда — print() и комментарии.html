Команда — print() и комментарии.

Начнем с написания программы«Привет! Меня зовут Наташа».
Эта программа будет выводить на экран текст: Привет! Меня зовут Наташа

Чтобы вывести что-то на экран, нужно дать компьютеру специальную команду. В языке Python такая команда — print().

Практически все языки программирования позволяют оставлять в коде комментарии. Они никак не используются интерпретатором и нужны исключительно для людей: чтобы программист оставлял пометки для себя и для других программистов. С их помощью добавляют пояснения, как работает код, какие здесь ошибки нужно поправить или не забыть что-то добавить позже.

Комментарии в Python начинаются со знака # и могут появляться в любом месте программы. Комментарий может занимать всю строчку.
